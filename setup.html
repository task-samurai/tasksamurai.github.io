Task Samurai Setup Guide

Step 1. Get your Task Samurai API key

The Task Samurai team will provide you with an API key. Store it somewhere as you'll need it later.

Step 2. Install the app in your repository

Click one of the icons below to install Task Samurai in your existing repository provider.

Step 3. (recommended, but optional) Connect your ticket provider

If you wish to receive recommendations for your tickets without typing anything you can connect your ticket provider to get file suggestions in 1 click only. If you don't connect it, you can still get recommendations for custom tasks as you type them on the browser extension.

Step 4. Install the IDE extension

Step 5. Have fun coding!

At this point you're ready to go! If you encounter any issues, please use the links below to let us know!
